# AdvancedPython
A repository consisting of various advanced python exercises.

## Project Structure

The project is organized into the following directories and files:

### `src/`
The source code directory.

*   **`main.py`**: Main entry point for the application.
*   **`example.txt`**: An example text file used for file I/O operations.
*   **`import_me.py`**: A module demonstrating import functionality.
*   **`code_order.py`**: A file showing the order of code execution.

#### `src/advanced_python/`
Contains examples of advanced Python concepts.

*   **`lambdas.py`**: Examples of lambda functions.
*   **`classses.py`**: Advanced class usage and object-oriented programming concepts.
*   **`containers.py`**: Usage of Python container data types.
*   **`exceptions.py`**: Advanced exception handling.
*   **`threadings.py`**: Multithreading examples.

#### `src/design_patterns/`
Implementations of common software design patterns.

#### `src/data_structures_algorithms/`
Implementations of data structures and algorithms.

*   **`complex_algorithms.py`**: Implementations of more complex algorithms.
*   **`interview_questions.py`**: Common coding interview questions and solutions.
*   **`big_o.py`**: Explanation of Big O notation and examples of time complexities.

##### `src/data_structures_algorithms/algorithms/`
*   **`search.py`**: Search algorithms (e.g., binary search, linear search).
*   **`sorting.py`**: Sorting algorithms (e.g., merge sort, quick sort).

##### `src/data_structures_algorithms/data_structures/`

###### `lists/`
*   **`node.py`**: Base Node class for linked lists.
*   **`linked_list.py`**: Singly linked list implementation.
*   **`doubly_linked_list.py`**: Doubly linked list implementation.
*   **`circular_linked_list.py`**: Circular linked list implementation.

###### `trees/`
*   **`binary_tree.py`**: Binary tree implementation.
*   **`binary_search_tree.py`**: Binary search tree implementation.

###### `queues_deques_stacks/`
*   **`deque.py`**: Double-ended queue implementation.
*   **`queue.py`**: Queue implementation.
*   **`stack.py`**: Stack implementation.
